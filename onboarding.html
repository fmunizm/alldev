👋 Bem-vindo ao AllDev GPT

Este assistente GPT é capaz de gerar funcionalidades completas para seus sistemas, fazer deploy em VPS, executar testes e monitorar sua aplicação com base apenas em comandos em linguagem natural.

🚀 Como usar

  1. Acesse chat.openai.com/gpts/editor e importe a action com a URL abaixo:
    2. https://fmunizm.github.io/alldev/openapi.yaml
  2. Informe a URL do repositório GitHub onde deseja que o GPT atue.
  3. Forneça seu token pessoal do GitHub (PAT) com os seguintes escopos: repo, workflow, user. Certifique-se de copiar e colar o token quando solicitado pelo assistente.
  4. Pronto! Você pode agora pedir, por exemplo:
    • “Crie um sistema de login com Google”
    • “Adicione campo CPF no cadastro de cliente”
    • “Rode os testes automatizados”
    • “Faça deploy na minha VPS”

🛡️ Segurança

O token fornecido nunca é salvo fora da sua sessão. Você pode revogá-lo a qualquer momento em github.com/settings/tokens.
